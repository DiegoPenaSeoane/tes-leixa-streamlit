# TES Leixa v8.09 — Tablas personalizadas por fallo

## Qué corrige

La tabla ya no es solo genérica por tema.

Cuando fallas, la app muestra primero una tabla personalizada para esa pregunta concreta:

- concepto real entrenado
- dato clave del enunciado
- tu respuesta
- por qué te llevó al fallo
- respuesta correcta
- regla que debes memorizar
- cómo evitar repetirlo

Después, si procede, muestra una tabla de apoyo del concepto.

## Microtablas específicas añadidas

- ECG
- lesión medular
- Glasgow
- START

## Subir a GitHub

Sube:

- app.py
- requirements.txt
- README_v809_TABLAS_PERSONALIZADAS.md

Después:
- Commit changes
- Streamlit → Reboot app
- Ctrl + F5

Debe aparecer:

`Banco v8.09 · tablas personalizadas por fallo`
