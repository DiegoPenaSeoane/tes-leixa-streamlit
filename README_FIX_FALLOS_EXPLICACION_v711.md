# TES Leixa v7.11 — Fallos con explicación garantizada

## Qué corrige

Si fallas una pregunta, ahora la app fuerza siempre:

- respuesta correcta
- explicación del fallo
- por qué tu opción no era la correcta
- descarte de distractores si existe en el CSV
- regla de oro si existe
- error típico
- tip de examen
- mini-lección
- tabla/resumen si existe
- aprendizaje mínimo garantizado aunque falten campos

## Si aciertas

- no muestra bloque largo
- no entra en repaso de fallos
- solo confirma que queda dominada

## Subir a GitHub

Sube solo:

- app.py
- requirements.txt
- README_FIX_FALLOS_EXPLICACION_v711.md

Después:

- Commit changes
- Streamlit → Reboot app
- Ctrl + F5

Debe aparecer:

`Banco v7.11 · fallos con explicación garantizada.`
